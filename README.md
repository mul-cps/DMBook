# Jupyter Notebooks for the Data Modeling Book

This repository contains all Jupyter notebooks referenced in the book, along with the necessary datasets to run them.

Each notebook corresponds to a section or chapter in the book and is designed to be fully executable without requiring any additional data downloads.

## Repository Contents

- **Jupyter Notebooks** – all examples and exercises from the book.  
- **Data Files** – all datasets used in the notebooks.  
- **Used Images** – all images used in the notebooks.  

## Version Information

This repository is continuously updated and always contains the latest versions of all notebooks and associated data files.

## How to Use

1. Clone or download this repository.  
2. Open any notebook in Jupyter Lab or Jupyter Notebook.  
3. Run all cells to reproduce the results and examples from the book.
